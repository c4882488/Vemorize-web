# [Vemorize-web](https://github.com/c4882488/Vemorize-web)

單字背誦網站

-------- 

## 介紹

針對孩童製作一款簡易好上手的背誦單字小幫手，對於兒童不需要太複雜的功能，在背誦的時候也可以搭配我們製作的遊戲進行測驗，開發出app。

之後將App轉為電腦版，以班級形式做管理，在班級單字集與學生做CRUD，。
